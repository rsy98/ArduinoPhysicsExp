*Arduino介绍*

Arduino是一种风靡全世界的开放源代码以及硬件设计的软硬件电子设计平台，由意大利艾维里互动设计学院(Interaction
Design Institute Ivrea)的MassimoBanzi(该学院教师), David Mellis
(Banzi的学生)，
DavidCuartielles(当时在该学院做访问学者)等人开发完成，于2005年冬季问世。该平台主要由硬件和软件两部分构成。硬件部分是各种类型的Arduino控制板；软件部分是ArduinoIDE
(Integrated
DevelopmentEnvironment)，即计算机中的程序开发环境。具体使用时，先在IDE中编写程序代码，然后将程序烧录到Arduino电路板上的芯片，然后控制板按程序指令完成各种电路。

Arduino能通过各种各样的传感器来感知环境，通过控制灯光、电机等各种装置来反馈、影响环境。可以将各种传感器类比人的耳鼻眼等各种感官，各种LED、电机等类比人的双手。而Arduino控制板类比人的大脑，根据传感器的观测发布命令让各种电子元件如LED工作并给予反馈。Arduino已经在3D打印机、智能机器人、声光互动设计、智能穿戴等等领域都得到了广泛的应用。

为什么Arduino在全世界范围内越来越受到欢迎?这主要由于其具有以下几个显著的优点以及时代的特点。

- 简单易学，门槛较低。Arduino的软硬件对于初学者来说，极易掌握，同时有着足够的灵活性。不需要了解Arduino的内部硬件结构和各种专业的单片机知识，只需要有一定的编程基础，简单学习后，就可以快速地编写程序进行开发。

- 开源。Arduino的硬件原理图、电路图、IDE软件及核心库文件都是开源的，在开源协议范围内里可以任意修改原始设计及相应代码。

- 廉价。Arduino使用廉价的集成电路芯片，只需要几十块就能买到一块开发板。对于使用者来说，不需要付出很高的成本。

- 极高的自由度和可拓展性。Arduino在Windows、Linux和MacOS等操作系统上都能很好运行。控制板具有丰富的接口，可与非常多的扩展设备相连接以实现各种创意。

- 众多的使用者和良好的社区支持。Arduino有众多的开发者和用户，借助网络可以搜索到众多开源的示例代码、硬件设计。例如，可以在Github.com、Arduino.cc等网站找到Arduino第三方硬件、外设、类库等支持，更快更简单地扩展自己的Arduino项目。

- 互联网时代的特点。进入新世纪以来，各种新技术、新理念、新发现不断涌现，对学习者的精力和学习能力提出了新的挑战。网络时代的知识碎片化和快节奏特点使得学习者在掌握相关技术时很少有精力去深入长时间地学习。Arduino能有效地降低使用者的学习时间成本，让他们将更多的精力集中到想实现的创意上，也因而有更好的学习反馈和成就感。

*乐高积木介绍*

乐高公司1932年创办于丹麦比隆。其创始人Ole Kirk
Christiansen(1891--1958)是一位木匠，热衷于制作木制玩具。乐高公司的商标”LEGO”从1934年开始使用，来源于丹麦语”LEgGOdt”，意为”play
well”（玩得快乐）。目前市场上所能接触到的乐高积木是一种塑料的可拼插积木，这种塑料积木最早于1949年生产，目前在墨西哥、丹麦、匈牙利、捷克和中国嘉兴都有生产基地。其中中国嘉兴的生产基地于2016年底正式开业。乐高积木的发展历程简列如下。

- 1949年，第一块乐高塑料积木问世。两年后，穴柱连接原理的塑料积木投放市场。
- 1958年，在创始人的儿子Godtfred的主导下，乐高公司对拼插积木的设计进行了改进，使其拥有了更好的通用性和积木之间的牢固结合能力。这就是现在我们现在看到的乐高积木设计，同年该设计获得专利授权。
- 1963年，乐高在LEGO积木生产中改用ABS(Acrylonitrile
  ButadieneStyrene，中文译为:丙烯腈--丁二烯--苯乙烯)塑料，沿用至今。
- 1964年，第一种乐高套装发布。
- 1969年，面向幼儿且与标准乐高积木兼容的德宝(DUPLO)套装发布。
- 1977年，科技(Technic)系列积木发布。
- 1978年，人仔(Minifigure)发布。
- 1999年，第一代可编程积木Mindstorms发布。
- 2006年，第二代可编程积木------Mindstorms NXT发布。
- 2009年，可编程积木Mindstorms NXT 2.0发布。
- 2013年，第三代可编程积木 Mindstorms EV3发布。

*基于Arduino的物理实验介绍*

一般提到物理实验，联想到的场景是在物理实验室，那里有各种专业的黑箱集成的实验设备和其它实验元件等。这些教学或者科研器材多由学校等专业机构采购，成本较高。有实验意愿的物理爱好者在家里或者除实验室的其它地方只能进行一些趣味演示性的实验，而很少有机会进行各种测量性的物理实验。那么，有没有可能将物理实验搬到普通的教室或者家里的桌子，让我们能够用实验切身地感受物理的内涵和乐趣，而不只是停留在书面上的名词和公式?
本书用Arduino来展示这种可能性。用Arduino实施物理实验的主要优点在于其成本低廉和实验器材并不特别占地方，极大地降低了物理实验的门槛，而且测量的精度也能达到足够高的水平。此外，本书的物理实验也采用了乐高积木来辅助进行实验，读者可以注意到乐高积木非常适用于搭建各种实验用装置。乐高积木相当于一个可重复使用耗材的3D打印机，实验完毕之后可以拆卸搭建模型将积木用于其它实验。总之，Arduino能与乐高积木非常好地配合，成功实现各种实验设想。
